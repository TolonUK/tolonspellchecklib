Tolon SpellCheck
----------------

TolonSpellCheck.h - The public API for the TolonSpellCheck library.
tscModule.h - The module-wide object which hosts all the TSC sessions.
tscSession.h - The session object which hosts a set of spell checking options, can persist these to/from a stream and initiates spell checking tasks.

tscDLL.cpp - The implementation of the public API methods. Primarily wrappers around the library objects.
tscModule.cpp - The implementation of the module-wide object.
tscSession.cpp - The implementation of the session object.
